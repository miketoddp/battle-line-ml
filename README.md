# Battle Line ML Project

Simulation and strategy optimization for Reiner Knizia's Battle Line using:

- Monte Carlo Tree Search (MCTS)
- SQLite for data storage
- PySpark for analysis and modeling

## Structure

- `src/game/` – core game logic
- `src/agents/` – decision-making agents
- `src/mcts/` – MCTS implementation
- `src/db/` – database schema and access
- `src/simulation/` – self-play engine
- `notebooks/` – analysis and modeling
- `data/` – local storage (not tracked)
